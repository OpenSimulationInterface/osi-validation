Detailed Description

The time of day at the location of the vehicle.

Requirements
field type repeated Requirements
seconds_since_midnight uint32 no Must be in range [0 , 86400)

The number of seconds [s] that have passed since midnight local time. Used for determining the current state of the circadian rhythm of a driver.

Note
Rename "local time" to local time zone.
No changes of daylight saving time or time zones are considered. So we should specify which time are we using.
This field intent is to describe fitness of the driver to operate a vehicle. Maybe we should move this field to Occupant. Or have another field hours awake.
